# AjjaniFit API

A cloud-based multi-tenant Gym Management System built with Laravel.

## Features

- Multi-Tenant Architecture
- Authentication (Laravel Sanctum)
- Gym Management
- Branch Management
- Member Management
- Trainer Management
- Membership Plans
- Attendance Tracking
- Workout Plans
- Diet Plans
- Income & Expense Management
- Salary Management
- Notifications
- Reports & Analytics

## Tech Stack

- Laravel 12
- PHP 8+
- MySQL
- REST API
- Laravel Sanctum

## Installation

```bash
git clone https://github.com/ajaypl2001/ajjanifit-api.git
composer install
cp .env.example .env
php artisan key:generate
php artisan migrate
php artisan serve
```
